Easy installation method:
1. open SE, press F12, click "Open plugin folder"
2. Close SE
3. Delete all TheLordOfTheRings files
4. Copy there the plugin version you want to use
5. Open SE and it's ready

also, you may want to keep in that folder 2 zip files, 
one for the new plugin and one for the one that is found
in the strange eons catalog. after downloading the online
version, make a zip with all the files.
just unzip here the package you want to use on step 4.
